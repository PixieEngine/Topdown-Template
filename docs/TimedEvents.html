TimedEvents#every(n, fn)

      

        
Execute fn every n frames.

        

          
Usage

          

player = GameObject()

          
          
player.include TimedEvents

          
          
doSomething is called every 4 seconds

          
          
player.every 4, ->
  doSomething()
